üîåUnderstanding Providers

A provider in Terraform is a plugin that enables interaction with cloud platforms and services. Providers tell Terraform which APIs to use.

provider "aws" {
  region = "us-east-1"
}

resource "aws_instance" "example" {
  ami           = "ami-0123456789abcdef0"
  instance_type = "t2.micro"
}
